# Peggy

Product knowledge base for Fanatics Collect — strategy docs, frameworks, and PRDs that define how we build for collectors.

## What's in here

| Folder | What it covers |
|--------|---------------|
| **Knowledge/Quests/** | Collector quest strategy, frameworks, and the quest infrastructure PRD |
| **Knowledge/Games & Interactive/** | Games, quests, and programmes strategy; app architecture; structural model |
| **Knowledge/Programs & Quests/** | Building blocks shared across Topps programmes and collecting quests |
| **Knowledge/FanCash Trading Card/** | FanCash trading card proposal, redemption flows, and Figma board references |
| **Knowledge/Instant Rips/** | Digital packs PRD and review docs |

## How to contribute

See [CONTRIBUTING.md](CONTRIBUTING.md) for step-by-step instructions. The easiest way is to edit directly on GitHub — no terminal or git knowledge required.

## How it works

This repo is the **source of truth** for product knowledge. Changes here are published to Notion so the wider team can read them there. Do not edit these docs in Notion — those edits will be overwritten.

```
Contributors (GitHub) → Peggy repo (source of truth) → Notion (read-only mirror)
```
